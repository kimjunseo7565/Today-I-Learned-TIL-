## 정사각형의 길이를 입력하고 사각형을 출력하라
  - ? 10         // DartPad에서는 n = 10일 때의 출력으로 처리할 것
  ### 곽 찬 사각형
  ```
  void main() {
    String a = "";
    int n = 10;
    for (var i = 0; i < n; i++) {
      for (var j = 0; j < n; j++) {
        a += "*";
      }
      print(a);
      a = "";
    }
  }
  ```
  ### 테두리 사각형
  ```
  void main() {
    String a = "";
    int n = 10;
    for (var i=1;i<=n;i++) {
      for (var j=1;j<=n;j++) {
        if(i==1||i==n||j==1||j==n) {
          a += "*";
        } else {
          a += " ";
        }
      }
      print(a);
      a = "";
    }
  }
  ```
  ### / 표시 사격형
  ```
  void main() {
    String a = "";

    for (var i=1;i<=n;i++) {
      for (var j=1;j<=n;j++) {
        if(i==1||i==n||j==1||j==n||i==n+1-j) {
          a += "*";
        } else {
          a += " ";
        }
      }
      print(a);
      a = "";
    }
  }
  ```
