# 함수

## 형태
  - 특별한 기능을 하는 코드 묶음. 코드를 함수로 묶는 것을 모듈화라 한다.
  ```
  int f(int x) {
     return x + 10;
  }
  void main() {
     var result = f(10);
  }
  ```
  ※ C 계열 언어와  유사
***
## 함수와 메서드
  - 함수 : 모든 지역에서 접근 가능 함
  ```
  bool isEven(int number) {
     return number % 2 == 0;
  }
  ```
  - 메서드 : 클래스 안에서만 접근 가능하거나 인스턴스를 통해서만 접근 가능함
  ```
  class MyClass {
     bool isEven(int number) {
        return number % 2 == 0;
     }
  }

  …
  var myClass = MyClass();
  print(myClass.isEven(10);
  ```
